# germspeak
A simple imperative programming lanuage created that DOES NOT collect garbage.


# Documentation
### Keywords
- `germ(type arg1, type arg2)` : the functions for the language
- `int` : integer datatype
### Operations
- `+` : add two integers
- `-` : subtract integers
- `*` : multiply integers
- `/` : integer divide integers
